Content Recommendation System

Technologies: Python, FAISS, SQLite, FastAPI

The system leverages embedding-based similarity search using FAISS to efficiently retrieve relevant content, with user and content data stored in a SQLite database. A FastAPI backend exposes the recommendation logic through RESTful endpoints, demonstrating the integration of data science techniques with backend engineering to deliver scalable, user-centered AI solutions.
